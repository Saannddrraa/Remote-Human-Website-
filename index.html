Our innovative Avatar System enable an operator to see, hear, and interact within a remote environment. VR robot avatar is a project which allow a user to experience the world from a remote robot's perspective using Virtual Reality HMD (Head-mounted display).

Benefits of HMD

VR robot platform will allow users to teleoperate over long distance and in real-time remotely. Users will be able to control their point of view in the scene using virtual reality HMD and haptic devices, increasing environmental awareness, and enabling fine-grained control.
